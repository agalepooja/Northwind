#  Northwind SQL Practice Project

##  Project Overview
This project contains a collection of **SQL queries** written on the **Northwind sample database** to solve real-world business problems.  
The queries range from **basic data retrieval** to **advanced analytical queries** using joins, aggregations, grouping, and filtering techniques.

The main goal of this project is to **strengthen SQL skills** and demonstrate the ability to analyze business data using structured queries.

---

##  Objectives
● Practice SQL on a real-world business database  
● Improve data retrieval and analysis skills  
● Perform business-focused analysis using SQL queries  
● Demonstrate SQL proficiency for data analyst roles  

---

##  Skills & Concepts Used
● SQL (MySQL / SQL Server compatible syntax)  
● SELECT, WHERE, ORDER BY  
● Aggregate Functions (COUNT, SUM, AVG)  
● JOINs (INNER JOIN, LEFT JOIN)  
● GROUP BY & HAVING  
● Subqueries  
● String matching using LIKE  
● Date filtering  
● Business-focused data analysis  

---

##  Database Used
**Northwind Sample Database**

### Tables Used
● Customers  
● Orders  
● OrderDetails  
● Products  
● Categories  
● Suppliers  
● Employees  
● Shippers  

---

##  Business Questions Covered
● Retrieve supplier and customer information  
● Find customers based on location  
● Identify top expensive and popular products  
● Analyze employee performance and sales revenue  
● Calculate total revenue per order and per category  
● Find frequent customers for loyalty programs  
● Analyze shipping partners usage  
● Perform product, category, and supplier analysis  

---

##  Example Insights Generated
● Top 5 most expensive and most sold products  
● Revenue generated by each product category  
● Employees ranked by total sales revenue
